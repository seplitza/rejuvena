## Общие рекомендации по генерации:

### Технические параметры для всех изображений:
- **Resolution:** Минимум 1920x1080 для горизонтальных, 1080x1920 для вертикальных
- **Format:** PNG или JPG (высокое качество)
- **Color Space:** sRGB
- **Upscaling:** Рекомендуется использовать встроенный апскейлинг Runware для финального результата

### Стилистическое единство:
- Все изображения должны иметь **мягкое, естественное освещение**
- Цветовая палитра: **тёплые, успокаивающие тона** (кремовый, золотистый, розовый, белый)
- Избегать агрессивных контрастов и холодных оттенков
- Общая атмосфера: **wellness, premium, доступность**

### Порядок генерации:
1. **Сначала:** ИЗОБРАЖЕНИЕ 1 (баннер) и ИЗОБРАЖЕНИЕ 2 (до/после) — самые важные
2. **Затем:** ИЗОБРАЖЕНИЕ 5 (портрет автора) — для доверия
3. **В конце:** ИЗОБРАЖЕНИЕ 3 и 4 (концептуальные) — можно заменить иконками, если нужно

### Alternative approach (если Runware не дает нужного результата):
- Для портретов: попробовать **Midjourney** через Discord API
- Для UI элементов: использовать **Figma** с AI плагинами
- Для медицинских иллюстраций: рассмотреть готовые **stock illustrations** (Shutterstock, Adobe Stock)

## Тестирование и отбор:

Для каждого промпта рекомендуется:
1. Сгенерировать **3-5 вариантов**
2. Выбрать лучший по критериям:
   - Соответствие теме
   - Профессиональный вид
   - Эмоциональный отклик
   - Техническое качество
3. При необходимости **доработать промпт** и повторить генерацию
